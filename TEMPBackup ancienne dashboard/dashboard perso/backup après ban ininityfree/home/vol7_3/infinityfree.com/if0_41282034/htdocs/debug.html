Architecture du site

index.html
Dashboard principal. Page HTML auto-suffisante servie statiquement. Charge toutes les données via fetch() côté client. Ne nécessite aucun traitement serveur à l'affichage. Modules intégrés : horloge Hanoï/Paris, météo Open-Meteo, AQI, musique SC+YT, RSS Le Monde, calendrier 4J.
api_calendrier.php
Moteur PHP de synchronisation iCloud. Récupère le flux webcal:// iCloud via HTTPS, effectue l'unfolding des lignes longues, filtre J-7 à J+21, gère les récurrences FREQ=WEEKLY, et écrit data.json. Appelé en tâche de fond par le dashboard toutes les heures.
data.json
Cache des événements. Fichier plat JSON régénéré à chaque appel à api_calendrier.php. Structure : metadata (date synchro, bornes, compteur) + tableau data[] trié par timestamp. Format date : DD/MM/YYYY HH:MM.
api_wallpapers.php
API de gestion des fonds d'écran. Endpoints : list (liste les fichiers dans /wallpapers/), upload (multipart POST, nommage auto wp51…), delete?file=, reindex (régénère wallpapers.json depuis le disque). Appelée depuis l'explorateur dans debug.html.
wallpapers.json
Index des fonds d'écran. Source de vérité pour le dashboard et le pomodoro. Chargé dynamiquement au démarrage par initPhotos() dans index.html et par pomodoro.html. Régénéré automatiquement par api_wallpapers.php lors des uploads/suppressions/réindexation.
/wallpapers/wp01…wp30.jpg
Photos locales. Hébergées sur le serveur. 5 absentes (wp08, wp18, wp20, wp27, wp28) car InfinityFree a refusé leurs téléchargements Unsplash. Pour les ajouter manuellement : nommer le fichier exactement wp08.jpg etc. et l'uploader dans /htdocs/wallpapers/, puis ajouter l'entrée dans wallpapers.json.
APIs externes
Open-Meteo (météo + AQI) — gratuit, sans clé.
allorigins.win — proxy CORS pour le flux RSS Le Monde.
SoundCloud Widget API — w.soundcloud.com/player/api.js.
Google Fonts — DM Mono, DM Sans, Syne.
Toutes ces APIs sont appelées côté client.
